# BuildX Analytics Project

## Dataset Selected

College Placement Records Dataset

## Why I Selected This Dataset

I selected this dataset to analyze student placement trends, salary distribution, and factors affecting campus placements. It provides meaningful insights that can help understand placement performance.

## Business Questions Answered

1. How many students were placed?
2. Which specialisation has the highest average salary?
3. What is the salary distribution among placed students?

## Tools Used

* Python (Pandas)
* SQLite
* Power BI
* GitHub

## Key Insight

Students who were placed received higher salary packages, and salary trends varied across different specialisations.
